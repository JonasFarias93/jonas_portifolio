# jonas_portifolio

Portfólio minimalista e responsivo para apresentar projetos (backend, frontend e games) e contato.

## Objetivo
- Ser rápido, simples e fácil de manter
- Centralizar projetos e links relevantes (GitHub / LinkedIn)
- Ser publicado via GitHub Pages

## Stack
- HTML + CSS + JavaScript (sem framework)

## Estrutura
- `index.html`: página principal
- `assets/css/style.css`: estilos
- `assets/js/app.js`: dados dos projetos e filtros
- `docs/*`: documentação curta (arquitetura, decisões e status)

## Rodar local
Abra `index.html` no navegador.

## Documentação
- `docs/ARCHITECTURE.md`
- `docs/DECISIONS.md`
- `docs/STATUS.md`
